🔒 Confidentialité & Flux de données

⚙️ Mode confidentiel

Anonymisation automatique des données personnelles
Quand activé, tous les emails, numéros de téléphone, adresses IP et numéros de sécurité sociale sont remplacés par [EMAIL] [TEL] [IP] avant tout envoi à l'API OpenAI. Les données originales restent intactes dans votre base de données.

📊 Flux de données — Ce qui reste chez vous vs OpenAI

✅ Reste sur votre serveur
  • 🗄️ Comptes utilisateurs & mots de passe (hachés)
  • 🎫 Tickets de support & historique
  • 📚 Base de connaissances (fichiers uploadés)
  • 🏢 Configuration de l'organisation
  • 📊 Métriques & analytiques
  • 🔑 Tokens d'intégration (GLPI, Slack…)
  • 🗂️ Logs d'audit complets
⚡ Envoyé à l'API OpenAI
  • 💬 Le message de l'utilisateur (anonymisé ✓)
  • 📄 Extrait KB pertinent — max 360 caractères
  • 🖼️ Capture d'écran si jointe (compressée)
  • 🎤 Audio vocal pour transcription (Whisper)
Garanties OpenAI API :
• Aucun entraînement de modèle sur vos données
• Rétention par défaut : 0 jour (suppression immédiate)
• DPA (Data Processing Agreement) disponible sur demande
• Conformité SOC2 Type II & ISO 27001

🇪🇺 Option hébergement Europe (Azure OpenAI)

Pour un hébergement des données IA en Europe (France/Irlande), l'assistant supporte nativement Azure OpenAI Service. Les modèles GPT-4o sont disponibles dans les régions France Central et West Europe. Azure OpenAI inclut un contrat RGPD & DPA Microsoft sans frais supplémentaires.
Variables d'environnement (Render)
OPENAI_BASE_URL=https://<resource>.openai.azure.com/openai/deployments/<deployment>
OPENAI_API_KEY=<azure-api-key>
OPENAI_MODEL=gpt-4o
