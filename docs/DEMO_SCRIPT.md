# Script Demo Client (10-12 minutes)

## 1) Introduction (1 min)
- "Assistant Support IT pour PME: moins de tickets, support plus rapide."

## 2) Parcours utilisateur (4 min)
- Connexion utilisateur.
- Choix probleme rapide + procedure.
- Demarrage chat -> solution.
- Ajouter une capture (drag & drop / copier-coller).
- "Ca n'a pas marche" -> recap -> creation ticket.
- "Mes tickets" + suivi.

## 3) Historique (2 min)
- Ouverture historique.
- Recherche et detail conversation.
- Reprendre la conversation.

## 4) Admin (3 min)
- Dashboard rapide.
- Test GLPI (message OK/KO).
- Invitation utilisateur + copie du token.
- Parametres entreprise (support + rappel tickets).

## 5) Conclusion (1-2 min)
- "MVP pret. Brancher IA + GLPI en production ensuite."
