## Comptes demo
- User: `DEFAULT` / `user@assistant.local` / `user123`
- Admin: `DEFAULT` / `admin@assistant.local` / `admin123`

## Script demo (10 min)
1. Connexion utilisateur.
2. Choisir un problème rapide.
3. Si non résolu, créer un ticket.
4. Voir “Mes tickets”.
5. Connexion admin et vérification des tickets.
