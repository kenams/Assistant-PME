# Pack Client (Remise)

## 1) Acces
- Admin: `http://<domaine>/app/admin/`
- Utilisateur: `http://<domaine>/app/user/`

## 2) Admin (3 etapes)
1. Renseigner GLPI (URL + tokens).
2. Renseigner support (email, tel, horaires).
3. Creer les utilisateurs / invitations.

## Logo client
- Par defaut: `app/assets/logo.svg`.
- Pour changer: remplacez `app/assets/logo.svg` par votre logo (format SVG recommande).
- Option rapide: ajouter `?logo=https://votre-logo.png` dans l'URL pour tester.

## 3) Utilisateur (simple)
1. Se connecter.
2. Decrire le probleme.
3. Si besoin, creer un ticket.
4. Suivre le ticket dans “Mes tickets”.

## 4) Maintenance
- Sauvegardes: quotidiennes.
- Mises a jour: planifiees.

## 5) Support
- Contact support: <a definir>.
